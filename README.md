# 🎨 Rejouice Clone

A visually stunning and interactive clone of the Rejouice website built using HTML, CSS, and JavaScript. This project focuses on smooth animations, modern UI design, custom cursor effects, video backgrounds, and immersive user experience.

## 🚀 Live Demo

https://abhinav00510.github.io/rejoice_clone/

---

## 📌 Features

- 🎬 Full-screen video background
- 🖱️ Custom animated cursor
- ✨ Smooth scrolling animations
- 🎭 Interactive hover effects
- 📱 Fully responsive design
- 🎨 Modern and minimal UI
- ⚡ Optimized performance
- 🔄 Dynamic page transitions
- 🎥 Rich multimedia integration

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GSAP (GreenSock Animation Platform)
- ScrollTrigger
- Locomotive Scroll
